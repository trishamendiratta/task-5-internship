# UrbanCart – E-commerce Web Application

UrbanCart is a responsive e-commerce web application built with **HTML**, **CSS**, and **JavaScript**.  
It allows users to browse products, filter by categories, search dynamically, and manage a functional shopping cart.  
The project is optimized for performance, mobile compatibility, and cross-browser usability.

## Features
- **Dynamic Product Listing** – Products are rendered dynamically using JavaScript.
- **Search and Category Filtering** – Quickly find products by name or category.
- **Functional Shopping Cart** – Add and remove products from a real-time cart.
- **Responsive Design** – Works seamlessly across desktop, tablet, and mobile devices.
- **Performance Optimization** – Minimized load times, optimized images, and efficient scripts.
- **Clean UI** – Simple, user-friendly interface for smooth navigation
